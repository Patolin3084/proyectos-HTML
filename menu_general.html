Transiciones y Animaciones


Transiciones: Las transciciones en Css permiten cambiar suavemente de un estilo a otro. Puedes especificar la duración, la funcion de tiempo y la demora para las transiciones.


Animaciones: Las animaciones en Css permiten definir movimientos mas complejos y secuencias de estilo en el tiempo. Utilizas keyframes para definir los estilos en varios puntos a lo largo de la duracion de la animacion.



Boxsizing

Boxsizing: La box-sizingpropiedad CSS nos permite incluir el relleno y el borde en el ancho y alto total de un elemento.



Flexbox

Flexbox: El módulo de diseño de caja flexible facilita el diseño de una estructura de diseño flexible y responsiva sin utilizar flotación ni posicionamiento.



Gradientes

Gradientes: Los gradientes CSS te permiten mostrar transiciones suaves entre dos o más colores específicos.



Grid

Grid: El módulo CSS Grid Layout ofrece un sistema de diseño basado en cuadrículas, con filas y columnas, lo que facilita el diseño de páginas web sin tener que utilizar elementos flotantes ni posicionamiento.



Mediaquery

Mediaquery: La mediaquery es una técnica CSS introducida en CSS3. Utiliza la @mediaregla para incluir un bloque de propiedades CSS solo si una determinada condición es verdadera.
